Einführung

Dieses Dokument enthält das gesammte Voodoo-Wissen ;o) zur Entwicklung von Software mit der Archimedes-Bibliothek. Die einzelnen Punkte werden in Form von speziellen Fragestellungen behandelt.

Wo liegt das Problem, wenn aus einem SortedListSubEditor immer das erste Element der Liste gelöscht wird, wenn man den Entfernenbutton drückt?

Mit hoher Wahrscheinlichkeit ist die equals-Methode des in dem SubEditor angezeigten Elements nicht korrekt implementiert oder falsch oder gar nicht im Archimedes-Modell konfiguriert. Sobald das geschehen ist, klappt's auch mit dem Entfernen :o)

 

Ich habe einen ListOwner auf einen ExtendedListOwner erweitert. Trotzdem bleiben deaktivierte Buttons im SubEditor sichtbar!

Prüfen Sie, ob Sie das Interface auch in der Kopfzeile der Klasse innerhalb der implements-Klausel auf den ExtendedListOwner erweitert haben ...
